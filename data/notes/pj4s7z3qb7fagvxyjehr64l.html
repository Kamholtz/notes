Tried to register two views with the same name

react-native-picker-select depends on an old version of React Native Picker Picker and so it needs to be forced to use the latest version to prevent having two version of the same control being registered

See: Invariant Violation: Tried to register two views with the same name RNCPicker · Issue #403 · lawnstarter/react-native-picker-select

NPM solution:

  "overrides": {
    "react-native-picker-select": {
      "@react-native-picker/picker": "2.2.1"
    }
  },

Yarn solution

  "overrides": {
    "react-native-picker-select": {
      "@react-native-picker/picker": "2.2.1"
    }
  },
